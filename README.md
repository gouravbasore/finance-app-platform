# Finance App Platform 🚀

This is a finance app I built using a full DevOps pipeline and cloud-native setup.

---

## 🔧 Tech Stack

* Backend: Node.js (Express)
* Database: PostgreSQL
* Containers: Docker
* Kubernetes: k3s
* CI/CD: Jenkins + Argo CD
* Ingress: Traefik
* Security: NGINX + Cloudflare

---

## 🏗️ How It Works

Code → GitHub → Jenkins → Docker → Argo CD → Kubernetes → Traefik → NGINX → Cloudflare

---

## 📁 Project Structure

finance-app-platform/
backend/
src/
Dockerfile
package.json

---

## 🚀 How to Run

cd backend
npm install
npm run dev

Open in browser:
http://localhost:3000/api/v1/health

---

## ✅ Features

* Budget CRUD APIs
* Health check APIs
* PostgreSQL integration
